This 2026 Edition is a cross-platform utility inspired by the classic SData Tool that allows users to "expand" the reported capacity of a USB drive by reformatting the device using custom block-count logic based on the mkdosfs(8) manual. Features include dual OS support for Windows (.exe) and Linux (ELF), a safety lock that blocks system drives (C: or SDA), physical capacity detection, and a 5-second progress animation, all compressed with UPX for maximum portability. For Windows, run CMD as Administrator using GraviUSB.exe -d [DriveLetter:] -s [Amount+Unit] -n [SingleLetterName]. For Linux, use sudo with sudo ./GraviUSB_Linux -d /dev/sdX -s [Amount+Unit] -n [SingleLetterName].
